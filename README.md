## 🚀 Featured Projects

<details open>
<summary><b>🧬 Antigen — Prompt-Injection Detection for LLMs</b></summary>
<br/>

An ML classifier that detects prompt-injection attacks in LLM / agent inputs, benchmarked against real-world adversarial data from **Gray Swan Arena**.

| | |
| :-- | :-- |
| **Stack** | Python · Machine Learning |
| **Domain** | LLM security · adversarial robustness |
| **Data** | Real-world adversarial prompts (Gray Swan Arena) |
| **Repository** | [BornaBoyafraz/Antigen](https://github.com/BornaBoyafraz/Antigen) |

Focused on the defensive side of LLM safety — turning messy adversarial inputs into a signal a system can actually act on.

</details>

<details>
<summary><b>⚙️ DRAKE — A Compiler for How Language Models Think</b></summary>
<br/>

A tiny smart compiler that watches how a language model "thinks" step by step, then builds faster shortcuts for it on the fly — using real **LLVM** compiler tech under the hood.

| | |
| :-- | :-- |
| **Stack** | Python · LLVM |
| **Domain** | LLM systems · compilers · inference optimization |
| **Repository** | [BornaBoyafraz/DRAKE](https://github.com/BornaBoyafraz/DRAKE---) |

Where classic compiler engineering meets modern model behavior — optimizing the reasoning path instead of just the code.

</details>

<details>
<summary><b>🎓 OpportunityOS — Opportunity Tracker for Students</b></summary>
<br/>

A full-stack application that helps students organize opportunities, deadlines, and applications in one place.

| | |
| :-- | :-- |
| **Stack** | Java · Full-stack |
| **Domain** | Productivity · student tooling |
| **Repository** | [BornaBoyafraz/OpportunityOS](https://github.com/BornaBoyafraz/OpportunityOS) |

Built to fix a real problem I felt myself — scattered deadlines and applications with no single source of truth.

</details>

<details>
<summary><b>🛡️ SafeWalk — Route Planning Optimized for Safety</b></summary>
<br/>

A walking route planner that optimizes for **safety instead of speed**. Built for Toronto using Toronto Police incident data, city streetlight coverage, and community reports.

| | |
| :-- | :-- |
| **Stack** | TypeScript |
| **Domain** | Geospatial · public-safety data |
| **Data** | Toronto Police incidents · streetlight coverage · community reports |
| **Repository** | [BornaBoyafraz/Safe-Walk](https://github.com/BornaBoyafraz/Safe-Walk) |

> Google Maps shows you the fastest route — SafeWalk shows you the one you'd actually take home at night.

</details>
